# Sprint 1: technical summary

This sprint was built on top of the work of [sprint 0](https://iss.signedsnow0.it/sprint0/) and aimed to build several cardinal components of the system, in particular:
* The main *cargoservice* actor
* The *sonar* subsystem
* The *IOPort* and *pushbutton* interfaces
* Concrete implementations of the following interfaces: *ISlot*, *IHold*, *IPosition*
